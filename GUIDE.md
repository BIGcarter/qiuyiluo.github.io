# 项目运行指南

## 1. 启动本地服务器
在终端（Terminal）中运行以下命令来启动 Jekyll 服务器：

```bash
bundle exec jekyll serve
```

## 2. 预览网页
命令运行成功后，终端会显示 `Server address: http://127.0.0.1:4000/`。
此时，打开浏览器访问以下地址即可预览您的网站：

👉 **http://127.0.0.1:4000/**

## 3. 实时修改
*   **修改内容**：您可以直接修改项目中的文件（如 `.md`、`.html`、`.yml`）。
*   **自动刷新**：保存文件后，Jekyll 通常会自动重新编译，您只需刷新浏览器页面即可看到更新。
*   **配置文件修改**：如果您修改了 `_config.yml` 文件，**必须**停止服务器（按 `Ctrl + C`）并重新运行 `bundle exec jekyll serve` 才能生效。

## 4. 停止服务器
在终端中按下 `Ctrl + C` 即可停止运行。
